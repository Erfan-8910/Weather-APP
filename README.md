# Climind

The Ultra-Glassmorphic SPA Harmonizing Real-Time Meteorology with Deep Cognitive Focus.

This project originally started as a simple, standard Weather App. However, as we reimagined its purpose, it evolved from a simple forecast viewer into a mindful daily companion. We renamed the project Climind (derived from the classical Latin Aether — the pure, clear upper air of the sky) to reflect this ultimate fusion. It represents the perfect harmony between the atmosphere outside and the mental clarity inside (Flow State).

[Try Climind Live](https://erfan-8910.github.io/Climind-W) • [Report Bug](https://github.com/Erfan-8910/Weather-APP/issues) • [Request Feature](https://github.com/Erfan-8910/Weather-APP/issues)

## Features

* **Unified Dashboard:**
    * Temperature & Real Feel
    * Humidity & Wind Speed
    * US Air Quality Index (AQI) with real-time health alerts
    * Hourly & 7-Day Forecast
* **Focus Mode (New!):**
    * Customizable Pomodoro Timer (Focus, Short Break, Long Break)
    * Custom minutes configuration with an interactive slider
    * Autoplay-safe Lofi Girl (Live) radio stream (with ad notice)
* **Dynamic Atmosphere:**
    * Background gradients change based on weather conditions (Clear, Rain, Snow, Storm).
    * Smooth, swirling color rotations around the viewport edges during active focus sessions.
* **Smart Theme Sandbox (New!):**
    * Custom Hex Color Picker Sandbox
    * Color palette serialization (Import & Export shareable codes)
* **Smart Geocoding:** Search for any city worldwide with auto-focus and error handling.
* **No API Key Required:** Plug and play! Uses the free Open-Meteo API.

## Tech Stack

* **HTML5** & Canvas (Particle System & Animation)
* **CSS3** (Grid, Flexbox, Backdrop-Filter, Keyframes)
* **JavaScript (ES6+)** (Vanilla State Engine, Web Audio API, Fetch API)
* **API:** [Open-Meteo](https://open-meteo.com/), [BigDataCloud Client Geocoder](https://api.bigdatacloud.net)
